Code Review Package

What this folder contains

- cheat-sheet.md — one-page quick reference for the 5-minute review (script, demo checklist, Q&A). 
- script.md — concise timed script with cues.
- architecture.mmd — Mermaid diagram: system architecture.
- auth-flow.mmd — Mermaid diagram: local + Google auth flow.

How to use

1. Open `cheat-sheet.md` and `script.md` in VS Code or your editor.
2. Open the mermaid files in a Mermaid-capable viewer (or paste the markup into https://mermaid.live/ for PNG/PDF export).
3. Print `cheat-sheet.md` or export it as PDF for a one-page reference during your presentation.
