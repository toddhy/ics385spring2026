Core files are App.jsx, IslandCard.jsx, and Header.jsx. The first contains island data in an array. It imports the components from header and islandcard, and uses .map() in a loop to create cards for each island. main.jsx loads the App component, and  index.html renders the page and links to main.jsx. Built with the help of Claude Haiku 4.5.

To run the program:
npm install
npm run dev
